# prompts/

Wiederverwendbare Prompt-Bausteine, mit denen Claude (Code, Cowork, Web)
markenkonforme Outputs erzeugt. Können in eigene Workflows kopiert oder
per Skill-Verweis konsumiert werden.

## Verfügbar

- `extraction.md`    – Erst-Befüllung des Brand-Systems aus sources/
- `newsletter.md`    – Newsletter-Erstellung mit GSS-Tonalität
- `cold-mail.md`     – Vertriebs-Cold-Mail
- `landing-hero.md`  – Hero-Copy für neue Landing Page
- `pitch-deck.md`    – Pitch-Deck-Outline
- `datasheet.md`     – Datenblatt-Strukturierung
