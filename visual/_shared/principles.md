# Übergreifende visuelle Prinzipien

> Status: 🟡 Wird befüllt.

## Was alle drei Marken visuell teilen
- Slate-basierte dunkle Hintergründe für digitale Outputs
- Sans-Serif-Hauptschrift Montserrat
- Mono für technische Labels (JetBrains Mono)
- "Made in Germany"-Signal sichtbar (Badge/Footer)
- Hairline-Trennlinien statt harter Borders
- Hochwertig, technisch, B2B – kein Consumer-Look

## Universelle Don'ts
- Keine Drop Shadows auf Logos
- Keine Stockfoto-Klischees ("lachende Menschen mit Headsets")
- Keine animierten GIFs in formellen Kontexten
