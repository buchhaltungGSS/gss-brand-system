# templates/

Wiederverwendbare Strukturen für nicht-Web-Outputs.

- `pptx/` – PowerPoint-Master mit Brand-Theme, Title-Slide, Content-Layouts, Closer
- `docx/` – Word-Vorlagen: Briefkopf, Memo, Angebot, Whitepaper
- `pdf/`  – PDF-Templates: Datenblatt-Layout, One-Pager
- `email/` – HTML-Templates: Signature, Newsletter-Wrapper, Cold-Outreach
- `social/` – Social-Card-Templates: LinkedIn, Twitter, OG-Image-Generator

Werden zunächst als Spec-Markdown angelegt (Layout-Beschreibung,
Token-Verweise). Echte Binary-Templates (.pptx, .docx) folgen in
einem zweiten Schritt.
